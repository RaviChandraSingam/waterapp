Architecture
System Overview
Full-stack application with web and mobile interfaces, backed by PostgreSQL.
🖥️

Web Application

React 18 + Vite 5 with responsive design. 8 pages covering all workflows — data entry, billing, reports, and administration.

⚙️

Backend API

Node.js + Express with 9 route modules. JWT authentication, role-based access, audit logging, and Excel import/export.

🗄️

Database

PostgreSQL 16 with 10+ tables: blocks, flats, readings, billing, config, users, audit logs, water sources, and cost items.

📱

Mobile App

Expo / React Native with 5 screens for on-site meter reading capture. Plumbers can enter data directly from each flat.

🐳

Docker Deployment

One-command setup with Docker Compose. Three containers: database, API server, and web frontend with hot-reload dev mode.

📊

Excel Integration

Upload existing Excel workbooks to import readings. Export formatted reports matching the original spreadsheet layout.

Access Control
Four User Roles
Each role sees a tailored interface with only the features they need.

Plumber

Field-level user. Captures meter readings per flat. Can view the dashboard and monthly records but cannot access billing, configuration, or user management.

✅ Dashboard   ✅ Monthly Records   ✅ Capture Readings

Accountant

Reviews and modifies readings, exports Excel reports, manages billing, and can upload data. Cannot manage users or system configuration.

✅ All Plumber access   ✅ Billing   ✅ Export   ✅ Upload

Water Committee

Full administrative control. Manages users, configures billing parameters (slab limits, multipliers, warning thresholds), and signs off on final billing.

✅ All Accountant access   ✅ Configuration   ✅ User Management

How It Works

1. Enter credentials — Username and password fields with auto-focus on username.
2. JWT token issued — Server validates credentials and returns a JSON Web Token (24-hour expiry).
3. Token stored — Saved in localStorage along with user details (name, role).
4. Session management — All API calls include the token. A 401 response automatically redirects to login.

6 Common Areas Tracked

Common area readings use 3-decimal precision for accurate billing separation between shared and individual usage.

Each reading shows who captured it, maintaining the same audit trail as individual flat readings.

These readings are factored into the billing formula — common area water is subtracted from total input before calculating per-flat costs.

Key Features

Previous month reference — Last month's closing reading shown as read-only context.
Whole-number step — Inputs increment by 1 (not 0.001) for faster field entry.
Inline edit — Click ✏️ on any existing row to modify without re-entering all data.
Full audit log — Every create/update is timestamped with the user, old value, and new value.
Data Integration
Excel Upload & Export
Seamlessly bridge between spreadsheet workflows and the web application.

📤 Upload Excel

Import meter readings from a filled-in .xlsx workbook — the same format used in the original manual process.

What gets imported:
✓ Flat meter readings (all 3 sequences)
✓ Water source readings (borewells + tankers)
✓ Cost items (salt, electricity, etc.)
✓ Common area readings
Upload stats — After upload, a banner shows exactly how many flats, blocks, sources, and cost items were imported.

📥 Export Excel

Download a fully formatted Excel workbook for any month. The export matches the original spreadsheet layout with multiple sheets:

Workbook contains:
📄 Summary sheet (water sources, costs)
📄 One sheet per block (A–E)
📄 Consumption & billing details

Why WaterApp?
Everything an apartment complex needs for transparent, fair water billing.
⚖️

Fair Billing

3-slab system ensures heavy users pay proportionally more. No flat subsidizes another.

🔒

Role-Based Security

4 roles with strict access control. Each user sees only what they need.

📋

Full Audit Trail

Every reading change is logged — who, when, and what was modified. Zero disputes.

📊

Excel Integration

Upload legacy spreadsheets. Export formatted reports. Bridge old workflows seamlessly.

⚙️

Fully Configurable

Slab limits, multipliers, warning thresholds, and tanker costs — all adjustable per month.

📈

Analytics Dashboard

Block-wise consumption, monthly trends, warning counts — all at a glance.
